## Component Inventory for Home Page (Lo-Fi)

- Top navigation
- Hero / welcome block
- Primary action buttons
- Quick stats cards
- Next-best-action cards
- Blueprint progress bars
- Study plan module
- Weak-topic module
- Recent activity feed/table
- Footer (desktop) / bottom nav (mobile)

---

## Recommended v1 Home Page Direction

Use **Variant A (Guided Dashboard)** as v1 because it best supports stressed learners with:

1. Immediate orientation (welcome + clear CTAs)
2. Short path to resume studying
3. Explicit remediation via weak-topic actions
4. Balanced motivational + analytical view
